The Basics

The Rubber Ducky is a microchip with a USB connector that pretends to be a keyboard. The attacker plugs it into a port on the target computer, and the Ducky runs a premade script to execute whichever commands the attacker desires. The computer sees these commands as legitimate input from the keyboard, which lets the attacker act as the user, bypassing most normal security.
