Defenses

Defenses agains the Ducky can take several forms, though each can be bypassed with some effort, so a combination of several methods is necessary to ensure security.

Physical Access

The Ducky requires physical access to the target machine, or specifically, a USB port on the target machine. It doesn't have to be consistent access, just long enough for the script to run and the attacker to remove or hide the Ducky.
