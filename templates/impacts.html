Impacts

The Rubber Ducky relies on the fact that, at some level, user input must be trusted. At a computer without this trust, no one would ever be able to do anything, so it would be useless. This means security is necessarily limited, and the Rubber Ducky exploits this by masquerading as user input. The impact of this is that security now requires mechanisms to verify user input, or otherwise prevent malicious input from ever entering the system. Antivirus software, firewalls, and network management are all bypassed, so another layer of security becomes necesary.

The Rubber Ducky, if programmed well, is very difficult to distinguish from user input. Effectively, anything a user could legitimately do or gain access to, the Ducky can as well. A typical exploitation of a computer starts with some security flaw and builds up to gaining control of a user account. The Ducky bypasses all of this and gives the attacker direct control of the user account, temporarily. Thus, the only limits on the damage are limits that apply to normal users, once the attacker has access to the machine.

Dangerous and hilarious Ducky script favorites:

  • Grabbing credentials
  • Changing SSH keys
  • Downloading and executing arbitrary files (just to show off)
  • Capturing all iMessages on OSX
  • Stealing all passwords saved on Google Chrome

All of these scripts are easily searchable on the web, and the Ducky itself only costs $45. Please watch out for your USB ports, they provide IMMEDIATE access to your computer.

Limitations of USB Security

At first glance, the opportunities to use a Rubber Ducky might seem limited, as it does require physical access to the target machine. After closer examination, however, the versatility of this system becomes clear. To begin with, people use a variety of computers to which almost everyone has access. How many people check the USB ports of a library computer they just started? How often have you left your computer running while you went to the bathroom, or not paid attention to it while you talked to someone else? USB ports are generally designed to be easy to access, in order to make them easier to use and improve the user experience, but this makes it easier for everyone to access, not just the intended user.
